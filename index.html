최초 관리자 계정이 없어요

아래 순서대로 Firebase 콘솔에서 관리자 1명을 먼저 만들어주세요. 이후에는 이 사이트 안의 관리자 화면에서 나머지 계정을 추가할 수 있어요.

  1. Firebase 콘솔 → Authentication → 사용자 추가에서 이메일/비밀번호로 계정 생성 (이메일은 아이디@handover.local 형식으로)
  2. 생성된 사용자의 UID를 복사
  3. Firestore Database → users 컬렉션 → 문서ID를 방금 UID로, 필드에 username(문자열), role="admin", active=true 추가
  4. 새로고침 후 아이디/비밀번호로 로그인

    반복업무 가져오기

    지난 스케줄 기록을 분석해 2회 이상 반복된 업무를 목록으로 보여드려요. 이미 사이트에 같은 이름의 업무가 있으면 자동으로 체크 해제됩니다. 필요 없는 항목은 체크를 풀거나 ✕로 완전히 제외하고, 분류/이름도 바로 수정할 수 있어요.
    일부 업무는 로그 간격을 분석해 매월/연1회 마감일이 자동으로 채워져 있어요(격월·분기 반복은 사이트가 아직 자동 일정으로 지원하지 않아 비고에만 남겨뒀어요). 실제 업무와 다르면 가져온 뒤 수정해주세요.
