최초 관리자 계정이 없어요

아래 순서대로 Firebase 콘솔에서 관리자 1명을 먼저 만들어주세요. 이후에는 이 사이트 안의 관리자 화면에서 나머지 계정을 추가할 수 있어요.

  1. Firebase 콘솔 → Authentication → 사용자 추가에서 이메일/비밀번호로 계정 생성 (이메일은 아이디@handover.local 형식으로)
  2. 생성된 사용자의 UID를 복사
  3. Firestore Database → users 컬렉션 → 문서ID를 방금 UID로, 필드에 username(문자열), role="admin", active=true 추가
  4. 새로고침 후 아이디/비밀번호로 로그인
